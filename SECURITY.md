# Security Policy

If you discover a security issue, do not open a public issue first.

Report it privately to the maintainer with:

- affected component
- reproduction details
- impact assessment
- suggested mitigation if available

Security-relevant areas include:

- MCP transport authentication
- provenance enforcement
- retry determinism
- operator escalation payloads
- secret handling in deployment paths

Please avoid posting proof-of-concept exploits publicly until the issue has been reviewed and addressed.
